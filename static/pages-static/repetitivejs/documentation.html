rptv

rptv is the core of repetitive.js, containing its every method. You can invoke it to get an array of DOM elements from document.querySelectorAll(). 


    rptv(css-selector);

Returns

  • Array [ ] - contains DOM elements

Download (only as full library)

  • source code minified (recommended)
  • source code unminified
  • npm

Examples


    rptv("body") // returns Array [ <body> ];
           
    rptv("body ul li").forEach(function(element){ // iterate over every element
               element.textContent = "help me";
           });

rptv.ajax

Sends an AJAX request.


    rptv.ajax(object);

Configuration object

Configuration object is passed as an argument to the rptv.ajax() function. The available properties are:

  • url - required, path you're trying to reach
  • onSuccess() - required, function called when the ajax call is successful, response object passed as an argument
  • type - "POST" or "GET", default is "GET"
  • timeout - miliseconds until the ajax call expires, default is 0 (doesn't expire)
  • data - data passed to XMLHttpRequest.send() method, default is null
  • onTimeout() - function called after the call expires, default is null
  • onError() - function called when there's an error

Response object

Response object is passed as an argument to the onSuccess function. Depending on the browser support, it can have different amounts of properties such as:

  • responseText
  • responseURL
  • responseXML
  • response

Download

  • source code minified (recommended)
  • source code unminified
  • npm

Examples

   
    rptv.ajax({
      url: "../package.json",
      onSuccess: function (resp) {
        console.log(resp.responseText);
      }
    });
    
    rptv.ajax({
      type: "GET",
      url: "../package.json",
      timeout: 6000,
      onTimeout: function(){ 
          console.log("timeout");
        },
      onError: function(){ 
          throw new Error("ajax error");
        },
      onSuccess: function (resp) {
        console.log(resp);
        console.log(resp.responseText);
        console.log(resp.responseURL);
        console.log(resp.responseXML);
        console.log(resp.response);
      }
    });

rptv.classes

Takes any DOM element/s and returns an object allowing you use methods on them, such as: adding, removing, toggling and checking if the element contains specified classes. It's like classList API but better and more browser compatible. 


    rptv.classes(DOM-element/s);

Returns

  • Object { } - allows you to chain different methods, only for add(), remove() and toggle()
  • Boolean { } - true if all elements contain specified classes, only for contains()

Methods

  • add(string) - adds all classes in a string to all specified elements
  • remove(string) - removes all classes in a string from all specified elements if they have them
  • toggle(string) - removes all classes in a string if the element has them, adds them it if it doesn't
  • contains(string) - returns true if all specified elements have all classes from the string, otherwise returns false

Download

  • source code minified (recommended)
  • source code unminified
  • npm

Examples

   
    const el = document.querySelectorAll("button");

    rptv.classes(el).add("btn-block btn-red other-class"); // adds these three classes to all button elements

    rptv.classes(el).remove("btn-red btn-big btn-something"); // removes these three classes from all button elements if they have them

    rptv.classes(el).toggle("btn-blue btn-gray"); // removes btn-blue and btn-gray class if the element has one, adds it if it doesn't

    rptv.classes(el).contains("btn-blue btn-block"); // returns true if all buttons have these two classes, otherwise returns false
           
    rptv.classes(el).add("btn btn-block").remove("some-class").toggle("btn-blue");
